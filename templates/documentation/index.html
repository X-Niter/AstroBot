Getting Started

AstroBot AI is a feature-rich Discord bot designed specifically for Minecraft modding communities, server management, and enhancing your Discord experience with cutting-edge AI-powered features. It combines the capabilities of multiple bots into one powerful solution.

Quick Start Guide

  1. Add AstroBot to your Discord server

    Click the "Add to Discord" button on our website or use this link.

  2. Set up permissions

    AstroBot needs certain permissions to function properly. Follow our permissions guide to ensure everything works correctly.

  3. Configure basic settings

    Use the !astro setup command to run through the basic configuration wizard.

  4. Explore features

    Use !astro help to see all available commands and features.

Discord Command Example

User: !astro help
AstroBot: Displays a list of all available commands and categories 

// Basic command structure
!astro [command] [subcommand] [arguments]

// Examples:
!astro help
!astro server status
!astro mod search create
!astro ai ask "How do I create a Minecraft mod?"
